---------------------------------------------------
Instructions for installing SEXTANTE-OpenJUMP Bindings
---------------------------------------------------

- Locate the folder where you have installed OpenJUMP (version 1.4 is needed). Usually it is named "openjump-1.4"
- Locate the extensions folder at [openJUMP folder]\lib\ext
- copy the binding files to that folder
- copy the SEXTANTE core files to that folder
