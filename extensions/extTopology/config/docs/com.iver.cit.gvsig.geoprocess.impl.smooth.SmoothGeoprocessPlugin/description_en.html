Smooth lines

Smoothes lines to improve their aesthetic appeal or cartographic quality. This is done by fitting a best-matching curve through the original vertices of the input geometries.

The types of curves implemented are:
  • Bezier
  • B-spline
  • Cardinal spline
  • Catmull-Rom spline
  • Cubic B-spline
  • Lagrange curve
  • Natural cubic Spline
  • NURBS
