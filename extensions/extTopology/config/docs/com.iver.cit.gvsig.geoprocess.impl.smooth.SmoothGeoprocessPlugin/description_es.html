Suavizado de líneas y polígonos

Suaviza polilíneas para mejorar su estética o valor cartográfico. Para ello, se calculan las curvas (de entre los diferentes tipos soportados) que mejor se ajustan a la secuencia de puntos de una polilínea o de un polígono.

Los diferentes tipos de curvas soportados son:
  1. BEZIER
  2. B-SPLINE
  3. CARDINAL SPLINE
  4. CATMULLROM SPLINE
  5. B-SPLINE CUBICO
  6. LAGRANGE
  7. SPLINE CUBICO NATURAL
  8. NURBS
